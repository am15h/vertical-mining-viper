# Turbo Charging Vertical Mining of Large Databases 

### Steps to run: (Windows)

1. `g++ main.cpp forc.cpp item_set.cpp vtv.cpp viper.cpp`

2. `a.exe input.txt`

### Steps to run: (Linux)

1. `g++ main.cpp forc.cpp item_set.cpp vtv.cpp viper.cpp`

2. `./a.out input.txt`
